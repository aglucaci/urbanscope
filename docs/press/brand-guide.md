# UMDB Press Kit

UMDB stands for Urban Microbiome Database.

This press kit contains the approved public-facing logo files and the core guidance for using them consistently in publications, articles, presentations, repository listings, and event materials.

## Primary Identity

- Preferred name: `UMDB`
- Expanded name: `Urban Microbiome Database`
- Preferred logo: `umdb-logo-primary-horizontal-transparent.png`

## Approved Assets

- `umdb-logo-primary-horizontal-transparent.png`
  Primary horizontal lockup for websites, slides, posters, and documents placed on colored or textured backgrounds.
- `umdb-logo-primary-horizontal-solid.png`
  Solid-background export for cases where transparency is not supported or where the mark needs a fully opaque canvas.
- `umdb-logo-icon-preview.png`
  Small-format preview useful for profile images, previews, or quick reference.
- `umdb-logo-variant-sheet.png`
  Internal comparison sheet showing alternate directions explored during logo development.
- `umdb-logo-original-concept.png`
  Earlier concept artwork retained for project history and archival context.

## Color Direction

Use the existing UMDB visual palette when pairing the logo with supporting materials.

- Deep navy: `#09111B`
- Panel navy: `#101A29`
- Teal accent: `#4EC5A8`
- Sky accent: `#73A7FF`
- Amber accent: `#FFCF70`
- Primary light text: `#EDF3FF`
- Secondary muted text: `#98A8C0`

## Usage Guidance

- Use the transparent horizontal logo whenever there is enough horizontal room.
- Keep generous clear space around the lockup. A good minimum is the height of the `U` in `UMDB` on all sides.
- Scale proportionally. Do not stretch, squash, crop, tilt, or recolor the mark.
- Avoid placing the logo on noisy photographic backgrounds unless contrast is clearly preserved.
- Do not add outlines, glows, drop shadows, or substitute fonts.
- Avoid placing the full horizontal lockup at very small sizes. For tiny placements, use an icon-safe asset or favicon treatment.

## Attribution

Suggested attribution line:

`UMDB (Urban Microbiome Database), public urban microbiome metadata resource.`

## Notes For Publications And Press

- Refer to the resource as `UMDB` on second reference after introducing the full name once.
- Keep the expanded form as `Urban Microbiome Database`.
- Do not refer to the current public-facing identity as `UrbanScope`.
