How To Use

Tips for navigating this larger interactive map.

Use fullscreen for exploration The fullscreen button expands the map surface so the choropleth is easier to inspect and present.
Filter by year range Narrow the release to a period of interest and compare how country representation changes over time.
Switch regional scope Change the map region to zoom the display toward a single continent-style view.
Right-click markers to load in explorer Right-click on any sample marker to open it directly in the BioProject Explorer for detailed inspection.
Use rectangle tool to select multiple samples Click the rectangle icon in the top-left toolbar to draw a selection area. All samples within the drawn rectangle will be loaded into the explorer.
The map uses resolved country labels for coverage views and BioSample latitude and longitude attributes for sample-coordinate views when present. It is intended for interactive exploration, not normalized epidemiological inference.
